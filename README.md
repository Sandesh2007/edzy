# edzy
 Eedzy: A Java-based Debian file installer inspired by Pop!_OS's Eddy. Simplify .deb file installations on Debian-based systems with an intuitive graphical interface. Version 0.1 now available, stay tuned for upcoming features!

![Screenshot](https://github.com/Sandesh2007/edzy/blob/main/Screenshot%20edzy.png)


## Installation

To install `edzy`, you can follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/Sandesh2007/edzy.git
    ```

2. Navigate to the `edzy` directory:

    ```bash
    cd edzy
    ```

3. Run the installation script:

    ```bash
    sudo python3 install.py
    ```
    This will install the `edzy` package on your system.
    
4. To run edzy :

    ```bash
    edzy
    ```
